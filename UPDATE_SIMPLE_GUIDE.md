# نظام التحديث البسيط - دليل الاستخدام

## 📦 كيفية استخدام نظام التحديث

### للمطور (أنت):

#### 1. رفع التحديث على MediaFire

1. ابني البرنامج:
```bash
npm run build:win
```

2. ارفع ملف `Elator POS Setup.exe` على MediaFire
3. احصل على رابط التحميل المباشر

**تحويل رابط MediaFire لرابط مباشر:**
```
الرابط العادي:
https://www.mediafire.com/file/xxxxx/ElatorPOS.exe/file

الرابط المباشر:
https://download####.mediafire.com/xxxxx/ElatorPOS.exe
```

أو استخدم: https://www.mediafire.com → اضغط Download → انسخ الرابط المباشر

#### 2. تحديث ملف update.bat

افتح `update.bat` وغيّر السطر:
```batch
set UPDATE_URL=https://رابط-التحميل-المباشر-هنا
```

#### 3. إرسال الملف للعميل

أرسل للعميل:
- ✅ `update.bat` (ملف التحديث)
- ✅ رسالة: "شغّل الملف ده عشان تحدّث البرنامج"

---

### للعميل:

1. **حمّل ملف `update.bat`**
2. **اضغط عليه دبل كليك**
3. **انتظر التحميل والتثبيت**
4. **هيشتغل البرنامج تلقائياً بعد التحديث**

## 🎨 مميزات النظام

✅ **بسيط جداً** - ملف واحد فقط
✅ **واجهة عربية** - كل الرسائل بالعربي
✅ **صوت تنبيه** - يشغل صوت لما التحديث يخلص
✅ **تلقائي** - يقفل البرنامج القديم ويشغل الجديد
✅ **آمن** - بيتأكد إن التحميل تم بنجاح

## 🎯 نصائح مهمة

1. **اختبر الملف قبل الإرسال**: شغّل `update.bat` على جهازك أولاً
2. **تأكد من الرابط المباشر**: لازم يكون رابط تحميل مباشر، مش صفحة
3. **احتفظ بنسخة احتياطية**: قبل ما العميل يحدّث
4. **اكتب رقم الإصدار**: في اسم الملف عشان تعرف أي نسخة

## 🐛 حل المشاكل

- تأكد من اتصال الإنترنت

- شغّل البرنامج يدوياً
- تأكد إن الملف المحمّل سليم

## 🎵 الصوت التنبيهي

البرنامج بيشغل صوت Windows الافتراضي بعد التحديث.

لو عايز تغيّر الصوت، غيّر السطر ده في `update.bat`:
```batch
powershell -c (New-Object Media.SoundPlayer "C:\Windows\Media\اسم-الصوت.wav").PlaySync();
```

أصوات Windows المتاحة:
- `Windows Notify System Generic.wav` (الافتراضي)
- `Windows Ding.wav`
- `Windows Notify.wav`
- `tada.wav`
