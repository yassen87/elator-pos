## 1. نظرة عامة على النظام {#overview}

### ما هو Elator POS؟

نظام **Elator POS** هو نظام إدارة متكامل لمحلات العطور يشمل:

✅ **إدارة المبيعات**: نقاط البيع (POS) مع طباعة فواتير
✅ **إدارة المخزون**: تتبع المنتجات والكميات
✅ **إدارة الموظفين**: الحضور والانصراف والرواتب
✅ **التقارير**: تقارير مبيعات ومخزون وأرباح
✅ **واتساب**: إرسال فواتير تلقائية للعملاء
✅ **التحكم عن بُعد**: متابعة المحل من الموبايل

## 2. متطلبات التشغيل {#requirements}

### 2.1 متطلبات الكمبيوتر (للمحل)

#### الحد الأدنى:
- **نظام التشغيل**: Windows 10 (64-bit)
- **المعالج**: Intel Core i3 أو ما يعادله
- **الذاكرة**: 4 GB RAM
- **التخزين**: 500 MB مساحة فارغة
- **الشاشة**: 1366x768 بكسل

#### الموصى به:
- **نظام التشغيل**: Windows 11 (64-bit)
- **المعالج**: Intel Core i5 أو أعلى
- **الذاكرة**: 8 GB RAM
- **التخزين**: 2 GB مساحة فارغة (للنسخ الاحتياطية)
- **الشاشة**: 1920x1080 بكسل (Full HD)

### 2.2 الأجهزة الطرفية

| الجهاز | الحالة | الملاحظات |
|--------|--------|-----------|
| **طابعة حرارية** | اختياري | لطباعة الفواتير (58mm أو 80mm) |
| **قارئ باركود** | اختياري | لمسح المنتجات بسرعة |
| **درج نقدية** | اختياري | يتصل بالطابعة الحرارية |
| **كاميرا ويب** | اختياري | لمسح QR Code للواتساب |

### 2.3 متطلبات الشبكة

- **اتصال إنترنت**: مطلوب للميزات التالية:
  - إرسال فواتير واتساب
  - التحكم عن بُعد (Mobile App)
  - التحديثات التلقائية
  - النسخ الاحتياطي السحابي

- **السرعة الموصى بها**: 5 Mbps أو أعلى

## 4. إعداد السيرفر المحلي {#server-setup}

### 4.1 تثبيت Windows Server

1. **تحميل Windows Server 2022**
   - من موقع Microsoft الرسمي
   - أو استخدام Windows 10 Pro كبديل

2. **التثبيت**
   - اتبع معالج التثبيت
   - اختر "Desktop Experience" للواجهة الرسومية
   - أنشئ حساب Administrator قوي

3. **التحديثات**
   ```powershell
   # تشغيل Windows Update
   Install-Module PSWindowsUpdate
   Get-WindowsUpdate
   Install-WindowsUpdate -AcceptAll -AutoReboot
   ```

### 4.2 إعداد الشبكة

1. **تعيين IP ثابت**
   ```
   الإعدادات → الشبكة والإنترنت → Ethernet
   - IP Address: 192.168.1.100
   - Subnet Mask: 255.255.255.0
   - Gateway: 192.168.1.1
   - DNS: 8.8.8.8, 8.8.4.4
   ```

2. **فتح المنافذ في الجدار الناري**
   ```powershell
   # فتح منفذ 5001 للـ API
   New-NetFirewallRule -DisplayName "Elator POS API" `
                       -Direction Inbound `
                       -LocalPort 5001 `
                       -Protocol TCP `
                       -Action Allow
   ```

3. **إعداد Port Forwarding في الراوتر**
   - افتح صفحة الراوتر (192.168.1.1)
   - اذهب إلى Port Forwarding
   - أضف قاعدة:
     - External Port: 5001
     - Internal IP: 192.168.1.100
     - Internal Port: 5001
     - Protocol: TCP

### 4.3 تثبيت البرنامج على السيرفر

1. **تحميل البرنامج**
   - من الرابط المرسل
   - أو من GitHub Releases

2. **التثبيت**
   - شغّل `ElatorPerfume-Setup.exe`
   - اتبع الخطوات
   - اختر "تثبيت للجميع"

3. **إعداد قاعدة البيانات**
   - البرنامج سينشئ قاعدة البيانات تلقائياً
   - الموقع: `C:\Users\[User]\AppData\Roaming\elator-pos\database.sqlite`

4. **تشغيل الخدمة**
   - البرنامج سيعمل كخدمة تلقائياً
   - للتحقق: Task Manager → Services → Elator POS

## 5. التحكم عن بُعد {#remote-control}

### 5.1 الخيارات المتاحة

#### الخيار 1: TeamViewer (موصى به)

**المميزات:**
✅ سهل الاستخدام
✅ آمن ومشفر
✅ يعمل من أي مكان
✅ نسخة مجانية للاستخدام الشخصي

**التثبيت:**
1. حمّل TeamViewer من: https://www.teamviewer.com
2. ثبّته على السيرفر
3. اختر "Unattended Access" (وصول دائم)
4. سجّل حساب TeamViewer
5. احفظ ID و Password

**الاستخدام:**
- ثبّت TeamViewer على موبايلك أو كمبيوتر آخر
- أدخل ID السيرفر
- أدخل Password
- ستتحكم في السيرفر كأنك جالس أمامه

**التكلفة:**
- مجاني للاستخدام الشخصي
- $49/شهر للاستخدام التجاري

---

#### الخيار 2: AnyDesk

**المميزات:**
✅ أسرع من TeamViewer
✅ استهلاك إنترنت أقل
✅ مجاني للاستخدام الشخصي

**التثبيت:**
1. حمّل AnyDesk من: https://anydesk.com
2. ثبّته على السيرفر
3. اضبط "Unattended Access"
4. احفظ AnyDesk Address و Password

**التكلفة:**
- مجاني للاستخدام الشخصي
- $10.90/شهر للاستخدام التجاري

---

#### الخيار 3: Chrome Remote Desktop

**المميزات:**
✅ مجاني تماماً
✅ يعمل من المتصفح
✅ من Google (موثوق)

**التثبيت:**
1. افتح Chrome على السيرفر
2. اذهب إلى: https://remotedesktop.google.com
3. اضغط "Set up Remote Access"
4. ثبّت الإضافة
5. اختر اسم للكمبيوتر و PIN

**الاستخدام:**
- افتح https://remotedesktop.google.com من أي جهاز
- سجّل دخول بنفس حساب Google
- اختر الكمبيوتر
- أدخل PIN

**التكلفة:** مجاني تماماً

---

#### الخيار 4: Windows Remote Desktop (RDP)

**المميزات:**
✅ مدمج في Windows
✅ سريع جداً
✅ مجاني

**العيوب:**
❌ يحتاج Windows Pro أو Server
❌ يحتاج إعداد Port Forwarding
❌ أقل أماناً

**التثبيت:**
1. على السيرفر:
   ```
   الإعدادات → النظام → Remote Desktop
   فعّل "Enable Remote Desktop"
   ```

2. على الراوتر:
   - Port Forward منفذ 3389
   - إلى IP السيرفر المحلي

3. للاتصال:
   - افتح "Remote Desktop Connection"
   - أدخل IP العام للراوتر
   - أدخل Username و Password

## 7. الصيانة والدعم {#maintenance}

### 7.1 النسخ الاحتياطي

#### تلقائي (موصى به)
```
البرنامج ينشئ نسخة احتياطية تلقائياً كل يوم
الموقع: C:\Users\[User]\AppData\Roaming\elator-pos\backups\
```

#### يدوي
```
الإعدادات → قاعدة البيانات → إنشاء نسخة احتياطية

سيتم حفظ ملف .sqlite
احفظه على فلاشة أو سحابة
```

### 7.2 التحديثات

#### الطريقة 1: تلقائي (في البرنامج)
```
البرنامج يفحص التحديثات تلقائياً كل 6 ساعات
عند توفر تحديث، سيظهر إشعار
اضغط "تحديث الآن"
```

#### الطريقة 2: يدوي (ملف BAT)
```
1. حمّل ملف update.bat
2. دبل كليك عليه
3. انتظر التحميل
4. سيعمل صوت عند الانتهاء
5. البرنامج سيشتغل تلقائياً
```

### 7.3 حل المشاكل الشائعة

#### المشكلة: البرنامج لا يفتح
```
الحل:
1. أعد تشغيل الكمبيوتر
2. تأكد من Windows Update
3. شغّل البرنامج كـ Administrator
4. أعد تثبيت البرنامج
```

#### المشكلة: الطابعة لا تعمل
```
الحل:
1. تأكد من توصيل الطابعة
2. تأكد من تثبيت التعريف
3. اختبر الطباعة من Windows
4. أعد اختيار الطابعة في البرنامج
```

#### المشكلة: قاعدة البيانات تالفة
```
الحل:
1. أغلق البرنامج
2. استرجع آخر نسخة احتياطية
3. الإعدادات → قاعدة البيانات → استرجاع
4. اختر ملف النسخة الاحتياطية
```
