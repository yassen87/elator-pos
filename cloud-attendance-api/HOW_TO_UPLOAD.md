# ☁️ تشغيل نظام الحضور أونلاين (PHP)

ملف `api.php` هو كل ما تحتاجه لتشغيل النظام على استضافتك.

## الخطوات

1.  **الدخول للاستضافة**: افتح لوحة تحكم موقعك (cPanel أو Hostinger hPanel).
2.  **مدير الملفات (File Manager)**: اذهب إلى مجلد `public_html` (أو المجلد الفرعي الذي تريده).
3.  **إنشاء مجلد جديد**: سمّه مثلاً `attendance`.
4.  **الرفع**: قم برفع ملف `api.php` داخل هذا المجلد.
5.  **التجربة**: افتح الرابط في المتصفح:
    `https://your-site.com/attendance/api.php?action=health`
    
    > يجب أن تظهر رسالة: `{"success":true,"message":"Cloud API is working! 🚀"}`

## ما الخطوة التالية؟

بعد التأكد من أن الرابط يعمل، سنقوم بوضعه في "إعدادات البرنامج" (على الكمبيوتر) لربط الموظفين، وفي "تطبيق الموبايل" لتسجيل الحضور.
